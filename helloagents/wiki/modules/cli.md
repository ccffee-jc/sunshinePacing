# 命令行模块（cli）

## 目的
提供 Linux/通用 CLI 入口，基于配置启动代理。

## 模块概述
- **职责:** 解析参数、加载配置、启动/停止服务、输出日志与状态
- **状态:** ✅稳定
- **最后更新:** 2026-01-21

## 规范

### 需求: Linux 配置启动
**模块:** cli
支持 `-config` 参数加载 YAML 并启动代理。

#### 场景: 无 GUI 启动
在 Linux 环境使用 CLI 启动代理服务。
- 预期结果1：无 GUI 依赖
- 预期结果2：可输出运行日志
